SUSTC.ME Subdomain A Records Updater

Client tool is developed by Python.

本工具是使用 Python 开发的SUSTC.ME子域名A记录更新工具。

Usage:
1. Install Python 2.7.x (URL: https://www.python.org/downloads/ )
2. use update_ip.bat in Windows or update_ip.py in *nix directly.
  1) Run without parameters then set your local ip.
  2) Run with an ip like 'update_ip 10.0.0.1' then set the ip you provided.
3. Add it to Startup if you want to do it everytime after boot.

使用方法:
1. 安装 Python 2.7.x (下载链接: https://www.python.org/downloads/ )
2. 在 Windows 下使用 update_ip.bat ；在 *nix 系统下直接使用 update_ip.py 。
  1) 不加参数运行会将 IP 设为本机 IP
  2) 使用一个IP作为参数运行会将 IP 设为你提供的这个 IP ，例如：'update_ip 10.0.0.1'
3. 如果想每次开机后自动更新，可以添加到系统的启动项。
